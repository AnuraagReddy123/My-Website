Machine Learning

  • Implementing an SVM from scratch

    Tech Stack: Jupyter, Python

    SVMs are widely used in a lot of applications in machine learning. They are quick and versatile, making them a staple in machine learning. However the internal working of an SVM is not so straightforward. Here I implemented an SVM from scratch, implementing the Lagrangian equation and also implemented the kernel trick in the code

  • A comparison between ARIMA and Deep Learning Models for Forecasting

    Tech Stack: Python, Pytorch, LSTM, CNN

    In this project, motivated by the rapid rise of deep learning techniques, I conducted a comprehensive comparison of traditional statistical models like ARIMA against modern deep learning models (LSTM, CNN) for time series forecasting. The main goal was to assess if classical models like ARIMA could still hold their ground against the cutting-edge deep learning architectures. This project provided valuable insights into the relative strengths and limitations of different forecasting techniques across various time series scenarios.

  • Sentiment Analysis of Reddit Travel Data

    Tech Stack: Python, PyTorch, BERT, spaCy

    I worked in a team to conduct sentiment analysis on the r/travel subreddit using Huggingface’s Reddit dataset, focusing on pre-Covid and during-Covid periods. After preprocessing the dataset, implementing spaCy’s lemmatization and resolving name variations for data quality and coherence, we applied the RoBERTa model for sentiment analysis and BERTopic for topic modeling, revealing insights into global travel patterns and sentiment shifts.

  • PRSUM: A pull request description generator

    Tech Stack: Python, PyTorch, LSTM, Attention

    PRSUM summarizes pull requests and generates pull request descriptions using multiple artifacts such as source code comments, commit messages, issue titles and graphs based on diffs of code

  • Tic-Tac-Toe using Reinforcement Learning

    Tech Stack: Python

    I created a Tic-Tac-Toe game using reinforcement learning techniques in python and programmed three distinct AI opponents, each employing its unique learning approaches: Value-based iteration, policy-based iteration, and Q-learning and trained and assessed these agents in playing Tic-Tac-Toe against human opponents.

Systems

  • MIPS Simulator

    Tech Stack: Python, MIPS

    I worked in a team to build a simulator for MIPS language to better understand assembly instructions and the instruction pipeline in computer architecture. We designed and implemented a pipeline structure and simulated a set of instructions for MIPS. We also implemented an LRU cache to see how it affects the number of stalls and cycles per instructions.

  • NHAN Compiler

    Tech Stack: C, Lex, Yacc, MIPS

    I worked on a team to design a new C-like language called NHAN, for which we built a compiler using C. It supported arrays, functions and recursive functions. It used an Abstract Syntax Tree(AST) as an intermediate representation and generated MIPS code as its output. This MIPS code could be run on qtSpim to generate output.

  • Student Marks Management CLI

    Tech Stack: C++, FreeBSD

    I worked on a team to design and implement a console-based C++ program for managing student marks data based on the kilo text editor for the FreeBSD kernel and Implemented statistical analysis, including average marks and highest/lowest marks. In addition we designed features for dynamic user and faculty management, supporting different users at the operating system level. (Admin, Students, Faculty) and utilized file operations to read and write data.
